# AquiTa.do - Presentacion de Proyecto (RD)

## 1) Resumen ejecutivo
AquiTa.do es una plataforma B2B2C para Republica Dominicana que combina:
- Directorio inteligente para discovery local.
- SaaS operativo para dueños de negocio.
- Marketplace para reservas, promociones y conversion.

Objetivo: convertir trafico local en ventas medibles para negocios dominicanos.

## 2) Problema que resolvemos
- Usuarios pierden tiempo buscando negocios confiables y con datos actualizados.
- Negocios pequenos y medianos no tienen un canal digital unificado para captar y gestionar clientes.
- Muchos leads se pierden por falta de seguimiento y baja trazabilidad.

## 3) Propuesta de valor
- Busqueda local por provincia, categoria e intencion.
- Perfil de negocio con reputacion, contacto directo y pruebas sociales.
- Panel de negocio con CRM ligero, reservas, promociones y analitica accionable.
- Integraciones para omnicanalidad (WhatsApp, pagos y eventos de conversion).

## 4) Segmentos y enfoque comercial
- B2C: personas que buscan servicios y comercios locales en RD.
- B2B: negocios locales que necesitan captar demanda y operar mejor.
- Modelo de ingresos: suscripcion SaaS + comision transaccional + publicidad interna.

## 5) Diferenciadores actuales
- Contexto dominicano real (lenguaje, categorias y flujo de contacto local).
- Multi-tenant y control por roles para operacion segura por organizacion.
- Base tecnica preparada para escalar (PostgreSQL + NestJS + React + observabilidad).

## 6) KPI base para operacion
- Conversion de busqueda a contacto.
- Conversion de contacto a reserva.
- Retencion mensual por organizacion.
- Ingreso mensual recurrente (MRR).
- Churn de negocios activos.

## 7) Equipo y ejecucion
- Product Lead: vision, roadmap y monetizacion.
- Backend: seguridad, API, multi-tenant, datos.
- Frontend: UX, conversion, rendimiento.
- QA/Operaciones: calidad de release, monitoreo y estabilidad.

## 8) Estado actual
- Plataforma funcional en web + API.
- Roles activos (`USER`, `BUSINESS_OWNER`, `ADMIN`) con permisos diferenciados.
- Endpoints de discovery, SaaS operativo, analytics y modulos de crecimiento.

## 9) Siguiente fase recomendada
- Fortalecer onboarding de negocios.
- Optimizar conversion de listados a contacto.
- Escalar cobertura por provincias prioritarias.
- Consolidar canal comercial con casos de exito y testimonios.
